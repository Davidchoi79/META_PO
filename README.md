## 🚀 시작하기

1. 저장소 클론
```bash
git clone [repository-url]
cd META_PO
```

2. 웹 서버에서 실행
```bash
# Python 3 사용시
python -m http.server 8000

# Node.js 사용시
npx serve .
```

3. 브라우저에서 `http://localhost:8000` 접속
